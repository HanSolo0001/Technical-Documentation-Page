Introduction

This documentation will provide you with some basics of HTML with some references at the end. HTML stands for Hyper Text Markup Language. HTML is used for the following:

  • the standard markup for creating web pages
  • the structure of the web page
  • telling the browser how to display content with the use of elements
Get Started

You do not need any previous experience to get started with HTML. All you need is a simple text editor like Notepad, and you can create a very basic website. HTML files have the .html extension attached to them.

There are many text editors out there like Visual Studio Code, a very popular tool for programmers and web developers. These editors are more advanced, have many useful features, and often are free.

Hello World

Let's create your first website. Open Notepad, and we will create Hello World. Hello World is where a lot of developers start out, regardless of the language their learning. HTML is not a programming language, but a structure language.

To create Hello World, simply type "Hello World." into your Notepad. Then save the file as index.html, or you can name it to whatever you like, as long as .html is in the extension name when you save the file. Index.html is most common used for the homepage/landing page of a website, but you do not need to do it that way.

HTML Elements

HTML elements are the tag names and content within those tag names. The following code block is a basic structure of an HTML page. The tag names are between the greater than (>) and less than ( <) symbols. All content put between these tags are part of the HTML element.

                                
<html>
<head>
</head>
<body>
</body>
</html>
HTML Attributes

HTML attributes provide additional information about HTML elements. Take the following tag and attribute:

                        
<a href="https://example.com"></a>

The a is the anchor tag, and the href is the attribute that links to an address. In this instance, the href points to a website.

Another example is the src attribute on the img tag, as shown below. The src attribute links to where the image is stored (the path), and will then imbed on the page using that link path. Notice the img tag is self closing, so it does not need the closing </img> tag.

                        
<img src="https://example.com/img1.jpg">
HTML Styles

The HTML styles are imbedded CSS values used to style your webpage. Styles are added by using the <style> tag. The following shows an example on how to imbed a style. The property is a CSS property. The value is a CSS value.

                        
<tagname style="property:value;">
HTML Formatting

HTML formatting is a way to give text special meaning and style. You can bold, italicize, and underline text with formatting. These are just a few of the formatting options you can do with text.

                        
                            
This is bold text. It is formatted by <b></b>.

                            
This is italicized text. It is formatted by <i></i>.

                            
This is underlined text. It is formatted by <u></u>.
HTML Comments

HTML comments are comments that you can make in your HTML, that will not show up on the webpage. These are used to help reference and notate important parts or your code. You make a comment by doing the following:

                        
                            <!-- Write your comments here -->

Remember! Comment are not displayed in the web browser on your page. They are there to document and reference your code.
